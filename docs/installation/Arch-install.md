# Manual

Run these commands:
```
sudo pacman -S git python3 python-pip
git clone https://github.com/The-All-Python-Project/SimplePythonBrowser
cd SimplePythonBrowser
pip install PyQt5 sip requests PyQtWebEngine
```
To run it:
```
python3 Simple_browser.py
```
