# Manual

Run these commands:
```
sudo dnf install git python3 python-pip
git clone https://github.com/Python-Browser/SimplePythonBrowser
cd SimplePythonBrowser
pip install PyQt5 sip requests PyQtWebEngine
```
To run it:
```
python3 Simple_browser.py
```
